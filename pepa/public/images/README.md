# Imagen del Login

Reemplaza el archivo `thermal-hero.png` en esta carpeta con tu imagen.

**Especificaciones recomendadas:**
- Formato: PNG o JPG
- Dimensiones: 500x400px o 600x500px (aspecto 5:4 o similar)
- Peso: < 300KB
- Contenido: Algo relacionado con procesos térmicos, esterilidad o validación

Cuando reemplaces la imagen, el login la mostrará automáticamente en la sección izquierda.
